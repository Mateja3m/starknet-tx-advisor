# SMDAK Starter dApp

Expo React Native starter dApp that demonstrates:
- wallet connect/disconnect
- mock DeFi swap flow
- mock NFT mint flow
- activity logs and transaction lifecycle

## Run

From repo root:
- `npm install`
- `npm run start:dapp`
- `npm run android:dapp`
- `npm run ios:dapp`

Deep link callback route:
- `smdak-dapp://callback`
