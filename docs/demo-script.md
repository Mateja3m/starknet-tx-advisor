# Demo Script

1. Start collector and web UI.
2. Open web app home (`/`).
3. Enter tx hash and click Analyze.
4. Show current status badge.
5. Show lifecycle timeline entries over time.
6. Show diagnostic explanation panel (meaning + next actions).
7. Show deterministic fee bump suggestion.
8. Toggle raw JSON view.
9. Export report JSON.
10. Open `/rpc/info` and show active RPC + fallback config status.
